## HOW TO CREATE (5 minutes per card):

### Using Canva (Free):
1. Go to canva.com
2. Click "Create a design" ‚Üí "Custom size" ‚Üí 1080 x 1080 px
3. Choose dark background template or create blank dark slide
4. Add text boxes with content above
5. Add simple icons/shapes from Canva library (optional)
6. Download as PNG

### Using Google Slides (Free):
1. Create new presentation
2. File ‚Üí Page setup ‚Üí Custom ‚Üí 1080 x 1080 px
3. Set slide background to dark color
4. Add text boxes with content above
5. File ‚Üí Download ‚Üí PNG

### Using PowerPoint:
1. New presentation ‚Üí Design ‚Üí Slide Size ‚Üí Custom
2. Width: 10.8", Height: 10.8"
3. Dark background, white/cyan text
4. Save as PNG
